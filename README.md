# Express-flights
---
A web browser App that shows a list of Airline flights and gives the user the ability to add new flights with their choice of airline and date.
---


## Technology's Used
---
 - Node JS
 - Express
 - MonoDB
 - Mongoose
 - EJS
 - CSS
 - Javascript
